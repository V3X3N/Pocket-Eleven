# POCKET ELEVEN

Welcome to the Pocket Eleven Mobile Game repository! This project is an ongoing mobile application development for a football manager game. The game is built using Dart and Flutter framework for the frontend, and it integrates Firebase for authentication (Firebase Auth) and data storage (Cloud Firestore). The development environment primarily utilizes VSCode and Android Studio.

## About the Project

This project aims to create an immersive and engaging football manager game experience for mobile devices. It allows players to manage their own football team, make strategic decisions, compete in matches, and lead their team to victory. The application leverages the power of Flutter for building beautiful and responsive user interfaces, while Firebase provides robust backend services for user authentication and real-time data storage.

## Features

- User authentication with Firebase Auth for secure login and registration.
- Integration with Cloud Firestore for real-time data storage and management.
- Player management: scout, sign, and manage players for your team.
- Team management: set formations, tactics, and strategies for matches.
- Competitions: compete in leagues, tournaments, and cup competitions.
- Match simulation: watch matches unfold with realistic gameplay simulation.
- Transfer market: buy and sell players to improve your team.
- Training and development: train players to improve their skills and abilities.

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/V3X3N/Pocket-Eleven.git
   ```

2. Open the project in VSCode or Android Studio.

3. Set up Firebase for the project and configure Firebase Auth and Cloud Firestore according to the documentation.

4. Run the application on an emulator or physical device.
